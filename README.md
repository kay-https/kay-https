1. Create an account at [wakatime.com](https://wakatime.com/)
2. Install the WakaTime extension in your editor
3. Copy your API Key
4. Add it as a GitHub secret: `WAKATIME_API_KEY`
5. The `update-wakatime.yml` workflow will do the rest!

### <img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-fire.svg" width="20" alt="Fire" /> GitHub Token (Recommended)

For advanced features, create a Personal Access Token:

1. GitHub Settings â†’ Developer settings â†’ Personal access tokens â†’ Generate new token
2. Select scopes: `repo`, `workflow`
3. Add as secret: `PAT_TOKEN`

This allows:
- Auto-merge Dependabot PRs
- Automatic stats updates
- Access to private repositories in statistics

## <img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-target.svg" width="28" alt="Target" /> FAQ

<details>
<summary><strong><img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-eye.svg" width="16" alt="Eye" /> How do I make the README appear on my profile?</strong></summary>

<br>

The repository must have the **same name as your username**. For example, if your username is `john`, create a repository called `john`. The `README.md` file in the root of that repository will be displayed on your profile.

</details>

<details>
<summary><strong><img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-code.svg" width="16" alt="Code" /> Do the automations work automatically?</strong></summary>

<br>

Yes! After pushing the files, go to `Actions` in your repository and click "Enable workflows". The automations will run at scheduled times or when you make commits.

</details>

<details>
<summary><strong><img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-fire.svg" width="16" alt="Fire" /> Do I need to pay anything?</strong></summary>

<br>

No! Everything is 100% free. The project uses GitHub Actions (free for public repositories) and free badge and stats services.

</details>

<details>
<summary><strong><img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-sparkles.svg" width="16" alt="Sparkles" /> Can I use it in commercial projects?</strong></summary>

<br>

Yes! The project is under the MIT license, which means you can use, modify, and distribute it freely, including in commercial projects.

</details>

<details>
<summary><strong><img src="https://raw.githubusercontent.com/DevYukis/.github/main/assets/emojis/purple-heart.svg" width="16" alt="Heart" /> How do I report bugs or suggest features?</strong></summary>

<br>

Open an [issue](https://github.com/DevYukis/.github/issues/new/choose) using one of the available templates. We have templates for bug reports, feature requests, documentation, and questions.

</details>
